# 워크샵 개요

이 워크샵은 UV와 Ruff를 주제로 사내 L&D 세션을 진행합니다. UV는 빠른 Python 패키지 관리 도구로, Ruff는 효율적인 린터와 포매터입니다.

## 목표
- UV와 Ruff의 기본 개념 이해
- 설치 및 사용 실습
- 프로젝트 관리와 코드 품질 향상

## 일정
- UV 세션: 소개, 설치, 명령어 등
- Ruff 세션: 소개, 설치, 설정 등

## 준비물
- Python 설치
- VSCode (Ruff 확장 추천)
