# Conventional Commits Examples

```
feat(auth): add OAuth2 login
fix(api): handle 404 for missing resource
docs(readme): add setup instructions
style(lint): apply ruff formatting
refactor(core): simplify validation
perf(cache): memoize heavy function
test(service): add unit tests for email sender
chore(ci): update pipeline
```

Breaking change:
```
feat(api)!: rename /users to /members

BREAKING CHANGE: clients must update endpoint
```
