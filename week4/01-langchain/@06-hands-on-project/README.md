# 실습 프로젝트: 종합 에이전트 구축

이 실습 프로젝트에서는 지금까지 배운 내용을 종합하여 실제로 동작하는 에이전트를 구축해봅니다.

## 프로젝트 목표

여러 도구를 사용할 수 있는 종합 에이전트를 만들어봅니다. 이 에이전트는:
- 날씨 정보를 조회할 수 있습니다.
- 수학 계산을 수행할 수 있습니다.
- 문서에서 정보를 검색할 수 있습니다.
- 사용자의 요청에 따라 적절한 도구를 선택하여 사용합니다.

## 프로젝트 구조

```
my-agent-project/
├── main.py              # 메인 에이전트 코드
├── tools.py             # 도구 정의
├── documents/           # RAG용 문서 디렉토리
│   └── knowledge.txt
└── requirements.txt     # 의존성 목록
```

## 단계별 구현

### 1단계: 도구 정의

`tools.py` 파일에 필요한 도구들을 정의합니다:

- 날씨 조회 도구
- 계산기 도구
- 문서 검색 도구

### 2단계: 에이전트 생성

`main.py`에서 여러 도구를 가진 에이전트를 생성합니다.

### 3단계: 테스트 및 개선

다양한 시나리오로 에이전트를 테스트하고 개선합니다.

## 실습 과제

1. **기본 에이전트 구축**: 위에서 설명한 도구들을 모두 포함하는 에이전트를 만드세요.

2. **RAG 통합**: 문서 검색 기능을 RAG 에이전트로 구현하세요.

3. **에러 처리**: 도구 실행 중 발생할 수 있는 오류를 처리하는 로직을 추가하세요.

4. **대화 기록 관리**: 여러 번의 대화를 이어갈 수 있도록 대화 기록을 관리하세요.

5. **확장**: 새로운 도구를 추가하여 에이전트의 기능을 확장하세요.
   - 예: 날짜/시간 변환, 단위 변환, 번역 등
