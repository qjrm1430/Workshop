# 실습 프로젝트: LangGraph로 복잡한 에이전트 구축

이 실습 프로젝트에서는 LangGraph를 사용하여 복잡한 상태 관리와 제어 흐름을 가진 에이전트를 구축해봅니다.

## 프로젝트 목표

여러 단계의 워크플로우를 가진 에이전트를 만들어봅니다. 이 에이전트는:
- 여러 단계의 작업을 순차적으로 처리합니다.
- 조건에 따라 다른 경로로 분기합니다.
- 상태를 관리하고 추적합니다.
- 체크포인팅을 사용하여 장애 복구가 가능합니다.

## 프로젝트 구조

```
my-langgraph-project/
├── main.py              # 메인 그래프 코드
├── nodes.py             # 노드 함수들
├── state.py             # State 정의
└── requirements.txt     # 의존성 목록
```

## 단계별 구현

### 1단계: State 정의

복잡한 상태 구조를 정의합니다:
- 작업 목록
- 현재 단계
- 결과 데이터
- 메타데이터

### 2단계: 노드 구현

각 단계를 수행하는 노드 함수들을 구현합니다:
- 초기화 노드
- 처리 노드들
- 검증 노드
- 완료 노드

### 3단계: 그래프 구성

노드들을 연결하고 조건부 엣지를 추가합니다.

### 4단계: 체크포인팅 추가

영속성과 장애 복구를 위한 체크포인팅을 추가합니다.

## 실습 과제

1. **기본 워크플로우**: 선형 그래프로 간단한 워크플로우를 구현하세요.

2. **조건부 분기**: 상태에 따라 다른 경로로 분기하는 로직을 추가하세요.

3. **루프 처리**: 특정 조건이 만족될 때까지 반복하는 노드를 구현하세요.

4. **체크포인팅**: 체크포인터를 사용하여 상태를 저장하고 복원하세요.

5. **에러 처리**: 노드 실행 중 발생할 수 있는 오류를 처리하는 로직을 추가하세요.

6. **확장**: 새로운 노드와 엣지를 추가하여 워크플로우를 확장하세요.
